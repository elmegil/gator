# CLAUDE.md - hardware

## Purpose
KiCad hardware design files for the Hook's Gator trigger sequencer. The **current design** is in KiCad 6+ format (`.kicad_sch`/`.kicad_pcb`) at the root of this directory. Everything under `outdated/` and `backup*/` directories is superseded.

## Current Schematic Structure (KiCad 6)
Top-level: `trigger-seq.kicad_sch` — three sub-sheets:
- **clock.kicad_sch**: PIC16F15345 (U3), LM393 comparator (U4), pots, switches, clock I/O
- **switching.kicad_sch**: Two CD4051 mux chips, 2N3904 address/inhibit buffer transistors, TL072
- **outputs.kicad_sch**: TL072 output buffers, 16 step output jacks, LEDs

Supporting:
- **power.kicad_sch**: LM78L05 regulator (5V from +12V), ferrite beads, Eurorack power connector
- **trigger-seq.kicad_pcb**: Main PCB layout
- **trigger-seq-front-panel.kicad_pcb**: Front panel PCB
- **trigger-seq-panel.kicad_pcb**: Jack panel PCB

## Key ICs
| Ref | Part | Sheet | Role |
|-----|------|-------|------|
| U3 | PIC16F15345 | clock | Main sequencer controller |
| U4 | LM393 | clock | External clock input conditioning |
| U1, U2 | CD4051 | switching | 8-channel analog mux (step selection) |
| U5, U6 | TL072 | outputs/switching | Output buffering |
| U8 | LM78L05 | power | 5V regulator |

## Signal Architecture
Global nets connecting sheets:
- **rate/width/steps/swing** — pot wiper voltages to PIC ADC inputs (RB5/RB4/RB6/RB7)
- **R/S** — Run/Stop switch → PIC RA2
- **U/D** — Up/Down switch → PIC RA4
- **reset** — Reset input → PIC RA5
- **2x8/1x16** — Mode switch → PIC RC1
- **CKIN** — External clock jack signal
- **CKINBUF** — Clock input after LM393 conditioning (buffered/Schmitt-triggered)
- **pg / pgck / pgdt** — Pulse gate timing signals generated by LM393 comparator from the gate output; creates a trigger pulse in hardware without PIC involvement
- **adr1/adr2/adr3** — PIC address outputs (RC3/4/5) → 4051 A/B/C address pins
- **inh1/inh2** — PIC inhibit outputs (RC6/7) → 4051 chip enables
- **ckbuf** — PIC gate output (RC0) → output stage

## Hardware Notes
- NPN transistors (2N3904) invert address/inhibit lines between PIC and 4051s — firmware pre-inverts outputs to compensate
- The LM393 handles both external clock input conditioning (CKIN→CKINBUF) and pulse-gate generation (pg/pgck/pgdt) entirely in hardware
- `trigger-seq.csv` is **stale** — it reflects the old design (LM555 + 4029 counter). Regenerate from KiCad if a current BOM is needed
- `bom/ibom.html` is an interactive BOM; check whether it was regenerated for the current design

## What's in `outdated/`
Previous design revision using discrete logic (LM555 astable, 4029 binary counter, LM311 comparators instead of PIC). Kept for reference only. Do not use as authoritative.
