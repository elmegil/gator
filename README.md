"Hook's Gator" trigger (gate) sequencer.

Controls:
- tempo
- gate width (need to state range)
- step count
- swing (50% - no swing, to 70%)
- run/stop switch and gate input
- forward/backward (need to test backward with swing) switch and gate input
- reset button and gate input
- clock in + clock out (external clock is reflected on the output and overrides internal tempo)
- 1x16 or 2x8 operation -- in 1x16 both outputs reflect the same
- each step has a switch that indicates if there should be a gate out on that step or not.
