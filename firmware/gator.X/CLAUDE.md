# CLAUDE.md - firmware/gator.X

## Purpose
MPLAB X project containing all firmware for the Hook's Gator trigger sequencer. Targets PIC16F15345 running at 8 MHz (OSCFRQ = 16 MHz config, but `_XTAL_FREQ` set to 8 MHz — verify before changing timing). Single source file; build artifacts are checked in.

## Key Files
- **main.c**: Entire firmware — init, ISR, ADC reads, main loop
- **Makefile / nbproject/**: MPLAB X build system; open in MPLAB X IDE or build via `make`
- **dist/default/production/gator.X.production.hex**: Flash image ready to program
- **dist/default/debug/**: Debug build artifacts

## Architecture

### Initialization
- `Init_PIC()`: Configures oscillator, disables unused peripherals (PMD registers), sets port directions/pull-ups, configures ADC
- `Init_interrupts()`: Sets up Timer 0 as the sequencer clock tick (1 ms resolution via `TM0COUNT = 225`, prescaler 1:32, Fosc/4 source)

### ISR (`__interrupt() ISR`)
Fires every ~1 ms from Timer 0. Handles all timing-critical sequencer logic:
- Increments `clock0`; compares to `period` to advance the step (`row`/`column`)
- Manages 2x8 (both columns) vs 1x16 (columns 1→2) modes and Up/Down direction
- Outputs the gate pulse via PORTC (bit 0 = clock/gate, bits 3-5 = 4051 address, bits 6-7 = chip select)
- Reset input (`RA5`) returns sequencer to start position
- While stopped: holds current LED on; while in reset: extinguishes LED

### ADC (`Acquire`)
Reads a 10-bit ADC value from the specified channel. Channels:
- `rate`  = RB5 → tempo (mapped through `curve[]` lookup table)
- `width` = RB4 → gate duty cycle
- `steps` = RB6 → sequence length (1–8 or 1–16)
- `swing` = RB7 → swing amount (offsets odd beats; **partially implemented** as of last commit)

### Speed Curve
`curve[1024]` — 6th-order polynomial lookup table mapping raw ADC (0–1023) to clock-tick period values (8–10199). Generated by Perl script embedded in comments. Provides logarithmic feel to the rate knob.

## Pin Map (PIC16F15345)
| Pin define | Port | Direction | Function |
|------------|------|-----------|----------|
| addrA/B/C  | RC3/4/5 | Out | 4051 address (step select) |
| inh1/inh2  | RC6/7 | Out | 4051 chip enable (active low via transistor inversion) |
| ckout      | RC0 | Out | Gate/clock output |
| runstop    | RA2 | In | Run=1 / Stop=0 |
| updown     | RA4 | In | Up=0 / Down=1 |
| reset      | RA5 | In | Active low (comment says "not found????" — verify) |
| mode       | RC1 | In | 2x8=0 / 1x16=1 |
| intclk     | RC2 | In | Internal clock (not ready to use) |
